# План: Сборка и установка PROD версии на iOS (iPhone Максим)

Этот план описывает процесс сборки приложения в конфигурации Release и его установку на физическое устройство.

## Шаги

1. **Подготовка окружения**
    - [ ] Переключить `.env` на `production` (API: `https://api.vedamatch.ru`)
    - [ ] Очистить старые артефакты сборки (опционально, если возникнут ошибки)

2. **Сборка приложения**
    - [ ] Выполнить `xcodebuild` в конфигурации `Release` для платформы `iOS`
    - [ ] Использовать `DerivedData` по пути `~/Library/Developer/Xcode/DerivedData/vedamatch-prod`

3. **Установка и запуск**
    - [ ] Установить сгенерированный `.app` на устройство `7077916C-5715-5713-AFFF-0DB2D4F159B0` (iPhone Максим) через `devicectl`
    - [ ] Запустить приложение

## Команда для запуска
`pnpm run ios:device:prod`

## Ожидаемый результат
Приложение будет установлено на телефон и запущено. Оно будет работать с продакшн-сервером.
