# Спецификация: VedaMatch UI Polish & Content

## 1. Обзор
Доработка интерфейса разделов знакомств и профиля для соответствия предоставленным макетам. Расширение набора тестовых пользователей для демонстрации всех режимов работы (Бизнес, Служение, Дружба).

## 2. Цели
- **UI Фильтров**: Сделать компактный и удобный интерфейс фильтрации (город, возраст в одну строку, выпадающие списки).
- **UI Профиля**: Привести внешний вид экрана редактирования в соответствие с дизайном (карточки, отступы, стили инпутов).
- **Контент**: Добавить 10+ разнообразных профилей для разных категорий поиска.

## 3. Требования к UI
- **Фильтры**:
    - Модальное окно с заголовком "Фильтры".
    - Поле "Город" (Input).
    - "Дата рождения" (Мин/Макс возраст) в одну строку.
    - Выпадающие списки (Tradition, Yoga Style, Guna, Identity) с белым фоном и скруглением.
    - Кнопки "Сбросить" (outline) и "Применить" (filled brown).
- **Профиль**:
    - Секции "Current City", "About Me", "Interests", "Tradition", "Yoga Style" с заголовками.
    - Многострочные поля для Bio и Interests.
    - Переключатель "Enable Dating Profile".

## 4. Тестовые Данные
- Добавить профили для:
    - **Бизнес**: Владельцы кафе, IT-специалисты, дизайнеры.
    - **Служение**: Организаторы фестивалей, пуджари, волонтеры.
    - **Дружба**: Музыканты (киртания), йоги.
