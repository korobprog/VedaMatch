# Спецификация: Библиотека Мудрости и Образование (v2.2)

## 1. Обзор
Этот трек направлен на создание персонализированной образовательной среды, которая адаптируется под духовную организацию (Матх) пользователя. Если пользователь в профиле указал принадлежность к **ИСККОН**, ему будут доступны специфические тренажеры Бхакти Шастры и Бхакти Вайбхава, соответствующие стандартам этой организации.

## 2. Ключевые Цели
- **Мульти-Матха поддержка**: Контент (тесты, вопросы) размечается по принадлежности к организациям.
- **Интеллектуальный фильтр**: Автоматическое отображение нужных тренажеров на основе поля `Affiliation` (или `Madh`) в профиле пользователя.
- **Всеобъемлющий Тренажер**: Полная база Q&A для популярных духовных программ.
- **Единый раздел Образование**: Консолидация чтения, тестов и медиа.

## 3. Архитектура Системы
1.  **Backend (Go)**:
    - Логика фильтрации: при запросе списка курсов бэкенд проверяет профиль пользователя и отдает только релевантный контент.
    - Модели данных поддерживают поле `organization_tag`.
2.  **Client Layer (React Native)**:
    - Экран "Образование" динамически подстраивает подборку курсов и цветовые акценты (опционально) под школу пользователя.

## 4. Пользовательские сценарии (Stories)
- **Как преданный ИСККОН**, я вижу в своем кабинете образования тренажер "Бхакти Шастры (ISKCON Standard)", разработанный специально для подготовки к официальному экзамену.
- **Как представитель другого Матха**, я вижу курсы своей школы, не путаясь в требованиях других организаций.
- **Как администратор**, я загружаю вопросы и ставлю тег "ISKCON", чтобы они были видны только целевой аудитории.

## 5. Модель Данных (Дополнения)
- `EducationCourse`: + `OrganizationID/Tag`.
- `ExamQuestion`: + `OrganizationID/Tag`.
- `User`: Использование существующих полей `Madh`/`Mentor` для определения фильтрации.
